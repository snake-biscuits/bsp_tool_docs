# `pixar.Usd`

Pixar's **Universal Scene Description** format[^org][^gh]

Widely adopted (NVIDIA[^nv], Apple[^ap], and more[^mr])

 * Used for composition (can be nested)
 * Supports multiple UV layers
 * Supports custom data
 * Complex shader schema (not parsed by Blender)
 * Supports Polygons


[^org]: [openusd.org](https://openusd.org/release/index.html)
