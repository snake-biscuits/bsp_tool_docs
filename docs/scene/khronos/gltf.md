# `khronos.Gltf`

The Khronos Group's "3D Asset Delivery Format"
Optimised for runtime sharing; "The JPEG of 3D"[^kh]

 * Supports multiple UV layers
 * Supports some custom data
 * Simple PBR shader schema
 * Triangles Only

[^kh]: Khronos Group: [glTF Overview](https://www.khronos.org/api/index_2017/gltf)
